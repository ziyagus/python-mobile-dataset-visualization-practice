# Технологическая практика: анализ и визуализация данных

Репозиторий содержит Python-проект для загрузки, анализа и визуализации табличного набора данных.

ID: 70221096  
Вариант датасета: 4

## Состав проекта

- `dataset.csv` — исходный набор данных;
- `dataset.py` — загрузка данных и создание `report.txt`;
- `data_scatter.py` — точечная диаграмма по числовым колонкам;
- `data_visual.py` — расширенная визуализация с разными типами графиков;
- `data_draw.py` — визуализация с режимом ручной разметки;
- `report.txt` — результат первичного анализа датасета;
- `requirements.txt` — список зависимостей;
- `screenshots/` — примеры тестового запуска.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python dataset.py
python data_scatter.py
python data_visual.py
python data_draw.py
```

Файл `dataset.csv` должен находиться в одной папке с Python-файлами.
