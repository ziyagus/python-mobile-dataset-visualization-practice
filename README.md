# Визуализация датасета мобильных устройств

Учебный проект по технологической практике.
В проекте реализованы Python-программы для загрузки CSV-датасета, анализа его структуры, построения графиков и ручной разметки визуализаций.

ID: 70221096
Вариант датасета: 4

## Описание проекта

В качестве исходных данных используется таблица с характеристиками мобильных устройств.
В датасете есть числовые поля, например размер экрана, объем оперативной памяти, объем накопителя, цена и рейтинг. Также есть категориальные поля: процессор и конфигурация камеры.

Проект позволяет:

* загрузить файл `dataset.csv`;
* получить первичную информацию о таблице;
* сформировать текстовый отчет `report.txt`;
* построить точечную диаграмму по числовым колонкам;
* использовать несколько типов графиков для числовых и категориальных данных;
* выбрать цветовую схему визуализации;
* нанести ручную разметку поверх графика;
* сохранить готовое изображение в PNG.

## Назначение основных файлов

| Файл              | Описание                                                                                    |
| ----------------- | ------------------------------------------------------------------------------------------- |
| `dataset.py`      | Загружает CSV-файл, создает `DataFrame`, выводит сведения о данных и формирует `report.txt` |
| `data_scatter.py` | Строит точечную диаграмму по выбранным числовым колонкам                                    |
| `data_visual.py`  | Добавляет разные типы графиков и выбор цветовой схемы                                       |
| `data_draw.py`    | Позволяет рисовать поверх графика и сохранять размеченное изображение                       |
| `dataset.csv`     | Исходный датасет варианта 4                                                                 |
| `report.txt`      | Текстовый результат первичного анализа                                                      |
| `screenshots/`    | Примеры работы программ                                                                     |

## Установка

Перед запуском нужно установить зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

Первичный анализ датасета:

```bash
python dataset.py
```

Точечная диаграмма:

```bash
python data_scatter.py
```

Расширенная визуализация:

```bash
python data_visual.py
```

Визуализация с ручной разметкой:

```bash
python data_draw.py
```

## Используемые библиотеки

* `pandas` — загрузка и анализ табличных данных;
* `matplotlib` — построение графиков;
* `Pillow` — работа с изображениями;
* `tkinter` — оконный интерфейс.

## Индивидуальные параметры

Для работы использованы параметры, рассчитанные по заданию:

| Параметр         | Значение  |
| ---------------- | --------- |
| Вариант датасета | 4         |
| Цветовая схема   | `magma`   |
| Маркер точек     | `d`       |
| Толщина кисти    | 9         |
| Цвет кисти       | `#160a60` |
